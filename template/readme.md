# 1 目的

作为工具的模板，以后新建工具，都可以基于该模板创建。log配置以及封装了配置文件加载，并提供了一些常用的库，希望以此为基础，不断完善这个模板，方便后续工具的开发。

# 2 特性
## 已有特性
* log系统：文件log系统，不同levellog
* 支持添加命令行参数
* 支持单元测试
* 支持性能分析

## 待实现特性
* UI框架：
* 网络：缺乏网络接口
* 数据库存储
* 性能分析整合火焰图
* slave客户端
* 多线程框架
* 多进程框架
